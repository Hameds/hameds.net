<p>این نوشته کوتاه به درد کسانی می‌خوره که از مک استفاده می‌کنند. برای استفاده از نرم‌افزارها روی مک هم می‌تونید از Mac App Store استفاده می‌کنید و هم می‌تونید مستقیم اون‌ها رو دانلود کنید.</p><p>اگر نرم‌افزار مربوطه در Mac App Store منتشر شده باشه موقعی که بروزرسانی نیاز داشته باشه، می‌تونید وضعیت رو ببینید و مدیریت دانلود بروزرسانی هم توسط اپ استور انجام می‌شه.</p><p>اما اگر نرم‌افزار رو به صورت مستقیم دانلود کرده باشید چطور؟ برای هر نرم‌افزار مدام نمی‌شه بروزرسانی‌ها رو چک کرد. یک نرم‌افزار <a href="https://github.com/mangerlahn/latest">اپن‌سورس</a> به نام <a href="https://max.codes/latest/">Latest</a> به شما کمک می‌کنه تا بروزرسانی اپ‌های خارج مک اپ استور رو مدیریت کنید.</p><figure class="kg-card kg-bookmark-card"><a class="kg-bookmark-container" href="https://max.codes/latest/"><div class="kg-bookmark-content"><div class="kg-bookmark-title">Latest</div><div class="kg-bookmark-description"></div><div class="kg-bookmark-metadata"></div></div><div class="kg-bookmark-thumbnail"><img src="https://max.codes/latest/img/app.png" alt=""></div></a></figure><p>این نرم‌افزار علاوه بر نمایش بروزرسانی‌ها، متن release note رو هم در صورت وجود نمایش می‌ده. اگر از مک استفاده می‌کنید، امتحانش کنید و نظرتون رو بنویسید.</p>
